# Eli7 — design explorations (11 Jul 2026)

Five visual-style prototypes for the Eli7 kid terminal (static, no functionality). Open `index.html` for the side-by-side gallery. Each variant shows three screens at 1024×600: welcome, conversation, and "thinking".

**Chosen working style: Pastel Kawaii** (`variant-2-pastel-kawaii.html`).

Kept for future reskin reference and to show Diana:
- `variant-1-cosmic-luma` — night-sky planetarium, glowing Luma star-mascot
- `variant-2-pastel-kawaii` — creamy kawaii, blushing cloud mascot  ← **chosen**
- `variant-3-comeau-playground` — joshwcomeau.com sensibility, gradient headline
- `variant-4-storybook-magic` — picture-book, expressive star-sprite
- `variant-5-bubbly-bold` — high-energy color blocks

Plan: implement one unified style (Pastel Kawaii), build substance, then restyle later — so keep the app theme-able (token layer) so a reskin is cheap.
